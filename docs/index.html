개인정보 처리방침

Rubato Manager

이 앱(이하 '본 앱')은 사용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다.

1. 개인정보의 수집 및 이용 목적

본 앱은 사용자로부터 어떠한 개인정보도 수집하거나 저장하지 않습니다. 본 앱의 주 기능은 사용자가 입력한 텍스트를 인텐트(Intent)를 통해 설정된 GitHub 저장소로 전달하는 것이며, 이 과정에서 개인 식별 정보를 서버로 전송하거나 기록하지 않습니다.

2. 수집하는 개인정보의 항목

본 앱은 다음과 같은 정보를 수집하지 않습니다.

  • 성명, 이메일 주소, 전화번호 등 개인 식별 정보
  • 기기 식별자 (ADID, IDFA 등)
  • 쿠키 및 접속 로그

3. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 외부에 제공하지 않습니다. 다만, 사용자가 직접 입력한 데이터는 사용자가 설정한 GitHub API를 통해 해당 저장소로 전송됩니다. 이 과정에서 개발자의 개인 API 토큰이 사용되며, 이외의 제3자에게 정보가 공유되지 않습니다.

4. 개인정보의 파기

본 앱은 정보를 수집하지 않으므로 파기할 대상이 존재하지 않습니다.

5. 이용자 및 법정대리인의 권리

이용자는 언제든지 앱을 삭제함으로써 서비스 이용을 중단할 수 있습니다. 수집되는 정보가 없으므로 별도의 열람이나 정정 요청이 필요하지 않습니다.
